Кратко о проекте:
--
BotSparrow - это игровой discord bot(бот, выполняющий определённые операции при определённых событиях, возникающих в дискорд серверах), написанный на python 3.7.5 при помощи Discord API и сторонних библиотек, а также программы Microsoft Visual Studio 2019.
Он включает в себя 4 игры. BotSparrow был создан для поддержания активности в дискорд серверах, а также просто для развлечения. Для взаимодействия с ним нужно вводить специальны команды, которые даст сам бот, просто впишите "!bothelp" в чат.

Чтобы собственноручно запустить и использовать бота, нужно:

1.Установить Discord.

2.Зарегистрироваться или войти в аккаунт и создать там ваш собственный сервер.

3.Пригласить на ваш сервер бота(ссылка для приглашения - https://discord.gg/m8Q7BG9Dh3).

4.Установить на компьютер python 3.7.5(64 bit) с помощью exe установщика в папке или с официального сайта.

5.Переместить с заменой папку Lib в папку с установленным пайтоном(у разработчика путь был следующим: C:\Users\user\AppData\Local\Programs\Python\Python37_64).

6.Запустить файл run.bat в папке.

7.Запустить Discord, зайти на ваш сервер и проверить, находится ли бот в сети, если да, то теперь им можно пользоваться.
--
P.S Даже если следовать этой иструкции, всёравно может произойти ошибка и бот не запустится, так как разработчик для создания бота пользовался Microsoft Visual Studio 2019!
